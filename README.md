## Installation
1. Clone the WestCoast-Education repository:
   ```bash
   git clone https://github.com/hedigardi/westcoast-education.git
    ```
2. Install the dependencies:
   ```bash
   npm install
    ```
3. Start the application:
   ```bash
   npx json-server --watch db.json
    ```
